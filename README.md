# Python Self-Study Journey

A day-by-day log of learning Python — building toward an **EE + Financial Analysis** specialization. This project applies each concept across multiple domains (Electrical Engineering, Finance, Software Engineering) to discover genuine interest before narrowing down.

## 🎯 Goals

- **Learn progressively**: Master Python fundamentals through small, daily projects.
- **Explore multiple domains**: Apply concepts to EE, Finance, and general Software projects.
- **Build discipline**: One concept → One day → One commit (consistent daily practice).
- **Specialize confidently**: Use breadth to identify the domain that resonates most, then go deep.
